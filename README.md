# Rent-A-Center

Rent-A-Center is a lease-to-own retail company providing furniture, electronics, appliances, and computers through its stores and Acima (formerly AcceptanceNOW) in-store kiosk program. Headquartered in Plano, Texas, Rent-A-Center operates thousands of stores across the United States, Mexico, and Puerto Rico. No public developer API is currently documented. The Acima platform provides a lease-to-own integration capability for partner retailers via a separate developer program.

**Human URL:** [https://www.rentacenter.com](https://www.rentacenter.com)

**Acima (Partner Platform):** [https://www.acima.com](https://www.acima.com)

---

## APIs

No public APIs are currently documented for Rent-A-Center. Partner integration is available through the Acima/Preferred Lease program for retail partners. The ePay portal at [https://epay.rentacenter.com](https://epay.rentacenter.com) provides customer self-service for existing lease agreement payments.
